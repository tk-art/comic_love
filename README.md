# 何ができるか
このアプリでは、家にいる時間が増えた今、オススメの漫画をシェアしていただくことを目的に作成しました
簡単な使い方としましては、自分がオススメしたい漫画を探していただき、自分のオススメポイントを添えて
投稿していただくというようなものです
投稿者が自分のオススメポイント以外に入力する手間を省くため、楽天APIからのデータを自動でフォームに入力されるようにしています
